# 📊 Clustering Analysis (K-Means & Hierarchical)

## 📌 Overview

This project demonstrates unsupervised learning techniques to group similar data points using clustering algorithms. It includes implementation of K-Means and Hierarchical clustering along with visualization of clusters.

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* SciPy

---

## 📂 Dataset

* Dataset containing numerical features used for clustering analysis
  *(Example: customer segmentation / income-spending dataset)*

---

## 🔍 Techniques Used

### 1️⃣ K-Means Clustering

* Applied K-Means algorithm to partition data into clusters
* Used **Elbow Method** to determine optimal number of clusters
* Visualized clusters using scatter plots

---

### 2️⃣ Hierarchical Clustering

* Performed Agglomerative clustering
* Built dendrogram to analyze cluster formation
* Identified natural groupings in the data

---

## 📊 Visualizations

* Scatter plots showing cluster separation
* Elbow curve for optimal K selection
* Dendrogram for hierarchical clustering

---

## 📈 Results

* Successfully grouped similar data points
* Identified meaningful patterns and clusters
* Compared performance of K-Means and Hierarchical methods

---
